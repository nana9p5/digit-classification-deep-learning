# Handwritten Digit Classification with Deep Learning

## 📌 Project Overview

This project was developed as part of my Master's in Artificial Intelligence & Big Data Analytics.

The goal is to classify handwritten digits (0–9) using a deep neural network.

## 🎯 Objective

Build a model capable of recognizing digits from image data.

## 🛠️ Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Scikit-learn
* Matplotlib

## 🔍 Approach

* Data loading and preprocessing
* Normalization of pixel values
* Train-test split
* Model building using a deep neural network
* Model training with callbacks
* Performance evaluation

## 🤖 Model

A fully connected neural network with:

* Multiple dense layers (512 → 256 → 128)
* Batch normalization
* Dropout for regularization
* ReLU activation
* Softmax output layer

## 📈 Results

The model achieves high accuracy on the test dataset.

## 💡 Key Learnings

* Building and training neural networks with TensorFlow/Keras
* Preventing overfitting using dropout and early stopping
* Working with image data and feature scaling
* Evaluating model performance

## 🚀 Future Improvements

* Use Convolutional Neural Networks (CNN)
* Hyperparameter tuning
* Data augmentation
